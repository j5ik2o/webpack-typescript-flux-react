# webpack-typescript-flux-react

Template Project by Webpack and TypeScript 2.0, Flux with React

## Frameworks/libraries that use mainly.

- flux
- react
- react-router
- lodash

## How to build

```sh
$ npm install
$ dtsm install
$ npm run build
```

## How to run

```sh
$ npm run server
```
open http://localhost:3000/
